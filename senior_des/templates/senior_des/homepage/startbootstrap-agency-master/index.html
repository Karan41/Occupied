Room Page

our project

Allows you to see the available rooms in the EER monitored by our Microcontrollers

Sensors

We used ____ sensors to be able to sense when people are in the room through body heat. These sensors send a boolean value? that gets read into the Microcontroller.

Microcontroller

We programmed Contiki OS onto all of our Microcontroller except the central hub, which has a Wifi module. All the Microcontrollers with Contiki will send its boolean value? into the central Microcontroller, which will send the information to the wifi via HTTP Post requests.

Server

Our backend SQLlite from Django will be waiting for POST requests that send information about the room's availability. The database will then update the room availability constantly based on the requests. The frontend will read the results of the database and modify itself, displaying the availability of the rooms
